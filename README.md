# G1 Driving Test Preparation Android App

Android application designed to help users prepare for the Ontario G1 driving test.

## Features

- Practice quiz with 180+ driving test questions
- Performance statistics using charts
- Google Maps integration to locate nearby test centres
- AWS backend for user login and registration
- Push notifications and social sharing

## Technologies Used

- Java
- Android Studio
- SQLite
- AWS Lambda
- AWS API Gateway
- DynamoDB
- Google Maps API
- MPAndroidChart

## Architecture

The application stores questions locally using SQLite while user authentication is handled through AWS cloud services.
